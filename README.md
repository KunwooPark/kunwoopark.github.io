# Academic Website with Quarto + GitHub Pages

This project is a minimal academic homepage for **Kunwoo Park** using **Quarto** and **GitHub Pages**.

## Files You Will Edit

- `index.qmd`: homepage
- `research.qmd`: research summaries
- `publications.qmd`: publication list
- `cv.qmd`: CV page
- `styles.css`: site colors and simple styling
- `files/`: put your CV PDF here
- `images/`: put your profile photo here

## Placeholders You Must Replace

Search the project for:

- `REPLACE_WITH_YOUR_EMAIL`
- `REPLACE_WITH_YOUR_GITHUB_USERNAME`
- `REPLACE_WITH_YOUR_LINKEDIN_ID`
- `REPLACE_WITH_YOUR_SCHOLAR_ID`
- `REPLACE_WITH_YOUR_CV_FILENAME`
- `REPLACE_WITH_`
- `profile-placeholder.jpg`

## Build Output

When you render the site, Quarto will create the final website inside:

- `docs/`

That `docs/` folder is what GitHub Pages will publish.
